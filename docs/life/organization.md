# 学生组织

学生组织分为系级和校级，包括运动队，艺术团，学生会，青志联等等。

学生组织也是有招新的，各位新生可以多留意一下，有的会宣讲，有的会贴海报。

以下是去年的部分校级组织的招新（还有个社团联合会，但我没找到），图片和介绍都来自于学生会和青志联qq官方账号。
