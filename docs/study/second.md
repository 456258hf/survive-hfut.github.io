# 第二课堂

第二课堂是学校为了学生全面发展要求的活动，可通过微信小程序的"第二课堂成绩单"查看。

毕业要求必修模块每个至少一星，选修至少有一个一星

:::tip
这个非常容易达到，学校有非常多的简单活动让你拿星星，所以根本不用愁
:::
