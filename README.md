# 🧱活在肥宣

Survive-HFUT

你的薰化路 301 号指南

## 运行

确保你已经安装了[NodeJs](https://nodejs.org/)和[pnpm](https://pnpm.io/zh/)

```sh
# 安装依赖
pnpm install

# 运行
pnpm run docs:dev
```

## 构建文档

```sh
pnpm run docs:build
```

## 更新所有包

```sh
pnpm run docs:update-package
```
